# LinkedInPlay

Automation toolkit for mastering LinkedIn Games.

LinkedInPlay automatically launches LinkedIn games using your existing
logged-in Microsoft Edge profile and plays them using fast, smart automation
logic. The project is modular, allowing multiple game solvers to be added
over time.

---

## Supported Games

- Zip (in progress)

---

## Planned Support

- Tango
- Queens
- Crossclimb
- Pinpoint

---

## Features

- Automatic Edge profile detection  
- Uses logged-in LinkedIn session  
- Smart game-specific solvers  
- Ultra-fast clicking engine  
- Modular architecture  
- Multi-game support (planned)  
- Human-like mode (planned)  

---

## How It Works

1. Detects default Microsoft Edge profile  
2. Launches Edge with logged-in session  
3. Opens LinkedIn Games  
4. Loads selected game  
5. Runs smart automation engine  

---

## Requirements

- Python 3.10+
- Microsoft Edge
- Edge WebDriver (matching your browser version)
